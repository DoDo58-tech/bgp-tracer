# 假连接验证流程图

## 当前流程（未优化）

```
┌─────────────────────────────────────────────────────────────┐
│  输入: 6560个假连接需要验证                                   │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
         ┌─────────────────────────────┐
         │  检查缓存                    │
         │  - 6520个命中 (已缓存)       │
         │  - 40个需要验证               │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  解析假连接为AS对            │
         │  "12345-67890;67890-11111"  │
         │  → {"12345-67890", "67890-11111"} │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  确定时间窗口                │
         │  过去一周:                   │
         │  2024-12-26 12:31:10        │
         │  到 2025-01-02 12:31:10      │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  找到2012个历史BGP更新文件   │
         └──────────────┬───────────────┘
                        │
        ┌───────────────┴───────────────┐
        │                                │
        ▼                                ▼
┌───────────────┐              ┌───────────────┐
│ 文件1         │              │ 文件2         │
│ (读取)         │              │ (读取)         │
└───────┬───────┘              └───────┬───────┘
        │                                │
        ▼                                ▼
┌───────────────┐              ┌───────────────┐
│ 提取Announcement│              │ 提取Announcement│
│ (可能有100万条) │              │ (可能有100万条) │
└───────┬───────┘              └───────┬───────┘
        │                                │
        │                                │
        ▼                                ▼
┌─────────────────────────────────────────────────────┐
│  对每条AS路径 (嵌套循环 - 性能瓶颈)                   │
│                                                      │
│  for as_path in announcements:                     │
│      segments = as_path.split()                    │
│      path_pairs = []                                │
│      for i in range(len(segments)-1):              │
│          path_pairs.append(f"{seg[i]}-{seg[i+1]}")│
│                                                      │
│      # 检查是否匹配任何假连接                        │
│      for fake_conn, as_pairs_set in fake_conns:    │
│          if as_pairs_set & path_pairs:              │
│              frequencies[fake_conn] += 1            │
│                                                      │
│  复杂度: O(路径数 × 假连接数)                         │
│  示例: 1000万路径 × 40假连接 = 4亿次比较            │
└─────────────────────────────────────────────────────┘
        │                                │
        └──────────────┬─────────────────┘
                       │
                       ▼
         ┌─────────────────────────────┐
         │  统计频率并更新缓存           │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  返回验证结果                │
         └─────────────────────────────┘
```

## 优化后的流程

```
┌─────────────────────────────────────────────────────────────┐
│  输入: 6560个假连接需要验证                                   │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
         ┌─────────────────────────────┐
         │  检查缓存                    │
         │  - 6520个命中                │
         │  - 40个需要验证               │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  优化1: 构建反向索引          │
         │  AS对 → [假连接列表]          │
         │  "12345-67890" → [fake_conn1]│
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  优化2: 提取所有涉及的AS      │
         │  {12345, 67890, 11111, ...} │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  找到2012个历史BGP更新文件   │
         └──────────────┬───────────────┘
                        │
        ┌───────────────┴───────────────┐
        │                                │
        ▼                                ▼
┌───────────────┐              ┌───────────────┐
│ 文件1         │              │ 文件2         │
│ (并行读取)     │              │ (并行读取)     │
└───────┬───────┘              └───────┬───────┘
        │                                │
        ▼                                ▼
┌───────────────┐              ┌───────────────┐
│ 提取Announcement│              │ 提取Announcement│
│ (100万条)      │              │ (100万条)      │
└───────┬───────┘              └───────┬───────┘
        │                                │
        ▼                                ▼
┌─────────────────────────────────────────────────────┐
│  优化3: 提前过滤 (减少99%的数据)                     │
│                                                      │
│  relevant = as_path.contains(any_fake_as)           │
│  filtered = announcements[relevant]                 │
│  # 只保留包含假连接AS的路径                           │
│  # 从100万条减少到1万条                              │
└─────────────────────────────────────────────────────┘
        │                                │
        ▼                                ▼
┌─────────────────────────────────────────────────────┐
│  优化4: 使用反向索引查找 (O(1)查找)                    │
│                                                      │
│  for as_path in filtered_paths:                    │
│      segments = as_path.split()                    │
│      for i in range(len(segments)-1):             │
│          as_pair = f"{seg[i]}-{seg[i+1]}"         │
│          if as_pair in reverse_index:            │
│              # O(1)查找，直接获取假连接列表          │
│              for fake_conn in reverse_index[as_pair]:│
│                  frequencies[fake_conn] += 1       │
│                                                      │
│  复杂度: O(过滤后路径数)                              │
│  示例: 1万路径 × 平均2次查找 = 2万次操作              │
│  (相比之前的4亿次，提升20000倍)                       │
└─────────────────────────────────────────────────────┘
        │                                │
        └──────────────┬─────────────────┘
                       │
                       ▼
         ┌─────────────────────────────┐
         │  统计频率并更新缓存           │
         └──────────────┬───────────────┘
                        │
                        ▼
         ┌─────────────────────────────┐
         │  返回验证结果                │
         └─────────────────────────────┘
```

## 性能对比

### 当前流程
```
文件数: 2012
每条文件路径数: ~1,000,000
总路径数: ~2,012,000,000 (20亿)
假连接数: 40
总比较次数: 20亿 × 40 = 800亿次
预计时间: 数小时
```

### 优化后流程
```
文件数: 2012 (并行处理)
每条文件路径数: ~1,000,000
过滤后路径数: ~20,000 (假设2%包含假连接AS)
假连接数: 40
总查找次数: 20,000 × 平均路径长度(5) = 100,000次
预计时间: 数分钟
```

**性能提升: 约8000倍**
